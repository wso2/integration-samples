# RAG Prebuilt Integrations

> **Note:** This feature is currently under development. The source code is contained in another private repository.

## Overview

This folder provides all the essential resources for working with RAG (Retrieval-Augmented Generation) prebuilt integrations, including:
- **Schemas**: Data models and interface definitions for integration.
- **Diagrams**: Visual representations of integration flows and architecture.
- **Setup Instructions**: Step-by-step guides to help you configure and deploy the integrations.

## Contents
- `config-schemas/` — JSON schema files containing the structure of how the config form will be rendered 
- `diagrams/` — Flow diagrams visually represents the sequential flow of data through different stages for each integration.
- `setup-instructions/` — Detailed setup and configuration instructions for each integration scenario.
