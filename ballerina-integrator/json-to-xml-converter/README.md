# JSON to XML Converter

## Description
A REST API that converts incoming JSON payloads into XML format and returns the transformed response.

## Usage Instructions

1. Run the integration locally using the **Run** button in WSO2 Integrator.
2. Use the **"Try It"** feature (auto popup) to send a `POST` request with a JSON payload and receive the XML response.

### Deploy on **WSO2 Cloud**

1. Use the **Deployment Options** to deploy this integration on **WSO2 Cloud** as an **Integration as API**.
2. Click **Deploy** and follow the instructions.
